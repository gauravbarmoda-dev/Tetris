# Tetris
A simple Tetris clone built using C++ and Raylib.

## Prerequisites
- C++ Compiler (e.g., g++)
- [Raylib](https://www.raylib.com/) library

## Building and Running
1. To build the game, run:
```bash
make build
```
2. To build and run the game, use:
```bash
make run
```
3. To clean the build artifacts, use:
```bash
make clean
```
